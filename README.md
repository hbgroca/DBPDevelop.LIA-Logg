# WIN 24 LIA Logg

## How-to
- Kör RUN_FIRST_TIME.bat innan första starten för att installera alla nödvändiga paket.
- Kör RUN_ALL.bat för att starta backend samt frontend servers. Webläsare öppnas automatiskt.
- OBS: All data sparas lokalt i mappen "~/public/" så gör backups regelbundet.
